# HUL Multiplicity Logic Simulation Sources

Simulation source for HUL Multiplicity Logic.

## Preparation

1. Click "Add Sources" in "Sources / Hierarchy" tab
2. Choose "Add or create simulation sources
3. Click "Add Files" and choose simulation sources you want to use

## Run Simulation

1. Right-click simulation source in "Simulation Sources/sim_1" under "Sources / Hierarchy" tab
2. click "Set as Top"
3. click "Run Simulation" in Flow Navigator (The left menu bar)
